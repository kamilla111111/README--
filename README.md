## Проекты

### 1. Ручное тестирование SauceDemo

Тестирование авторизации, корзины, сессии, локального хранилища через DevTools.

Артефакты: чек-лист, тест-кейсы, баг-репорты  
Инструменты: DevTools, Excel, Яндекс Браузер

Папка: saucedemo-testing


### 2. API-тестирование Demoblaze

Тестирование API интернет-магазина через Postman.  
Проверены эндпоинты: авторизация, товары. Найдены проблемы с статус-кодами и валидацией.

Всего запросов: 9  
Автотестов: 25  
Результат: 25 из 25 пройдено

Папка: demoblaze-api-testing


### 3. Ручное и API тестирование Demoblaze (полный цикл)

Полное тестирование демо-магазина: UI, API, безопасность, производительность.

Результаты:
- Чек-лист: 27 проверок
- Тест-кейсы: 27
- Найдено: 8 багов (2 критических, 5 серьёзных, 1 минорный)

Папка: demoblaze-full-testing


### 4. SQL-аналитика интернет-магазина

База данных: customers, products, orders, order_items.  
Использованы JOIN, GROUP BY, оконные функции, CTE.

Запросы:
- Топ-5 клиентов по тратам
- Самый популярный товар
- Товары без продаж
- Средний чек
- Самый дорогой заказ каждого клиента

Папка: sql-shop-analytics


### 5. SQL-аналитика доставки еды

База данных: customers, couriers, restaurants, orders, deliveries.  
Аналитика по курьерам, ресторанам, времени доставки.

Запросы:
- Выручка ресторанов
- Курьер с наибольшим количеством доставок
- Среднее время доставки
- Клиенты без заказов
- Самый популярный ресторан
- Самый быстрый курьер
- Рейтинг клиентов по сумме заказов

Папка: sql-food-delivery
